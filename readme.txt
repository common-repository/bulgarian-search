Bulgarian search is a plugin which enables you to have a Napred.BG search engine form for Internet search in a widget. 

== Description ==

[Napred.BG](http://napred.bg) is a popular Bulgarian search engine. If you want to enable your site's visitors to search the Bulgarian Internet directly from your blog, you can add a Napred.BG search widget with this simple plugin.

== Installation ==

The installation of Bulgarian search plugin is more than easy. You just need to follow these simple steps:

1. Upload `bulgarian-search` folder from zip archive to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Browse to Appearance->Widgets, and add the widget.
